Flaming Tires Auction

flamingtires.surge.sh

Github.com/ReadySetAuction

Flaming Tires Auction is a classic car auction site built to fill the needs of users searching for their dream car. Flaming Tires is a single page, multiple view application built using AngularJS JavaScript framework, compiled with TravisCI, and published through Surge. The main view displays upcoming auctions, with other views sharing auction consignments and televised auctions.

Web Develpment Tools Utilized:

  • HTML5
  • Jade
  • CSS3
  • SASS
  • JavaScript
  • AngularJS
  • JQuery
  • TravisCI

Weight Of The World News

weightofworldnews.surge.sh

github.com/WeightoftheworldNewsApp

Weight of the World News is a breaking news site built for users to keep up with the current news from a variety of news sites such as Aljazeera, BBC, Buzzfeed, CNN, and New York Times. Weight of the World News is a one page, multiple view application built using AngularJS JavaScript framework, compiled with TravisCI, and published through Surge.

Web Develpment Tools Utilized:

  • HTML5
  • CSS3
  • SASS
  • JavaScript
  • AngularJS
  • JQuery
